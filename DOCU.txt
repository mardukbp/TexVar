Implemetierte Funktionen in Latex Var CAS
1. Aufbau
	tVar.val: Wert der Variable als Nummer
	tVar.nameTex: Variablenname in Latex als string
	tVar.eqTex: Latex Darstellung der Gleichung um auf val zu kommen.
	tVar.eqNum: Latex Darstellung der Gleichung um auf val zu kommen. Statt Variablen Namen sind aber die val's eingefügt.

2. Variablen
tVar.New(_val, _nameText)
	Erzeugt eine neue Variable mit Latex Darstellung
	_val: Wert als Nummer
	_nameTex: Variablenname in Latex als string
	Beispiel:
		l_x2 = tVar:New(10,"L_{x2}")

tVar:fix(_nameText)
	Entfernt alle Gleichungen aus tVar und weist eine Latex Darstellung zu. 
	_nameTex: Variablenname in Latex als string
	
tVar:bracR()
	Fügt der Variablen Gleichung Runde Klammern hinzu. Dient nur der Latex Darstellung

tVar:printFull()
	Komplette Ausgabe aller Gleichungen
tVar:printHalf()
	Wie Full nur ohne eqNum
tVar:printVar()
	Variablenname und Wert

3. Mathematische Operationen
3.1 Standard
+,-,(-),*,/,^
3.1 Erweitert
tVar.sqrt(tVar,n)
	nte Wurzel aus tVar
	
4. Allgemeine Operationen
print(tVar) gibt val zurück

5. Konstanten
tVar.PI
