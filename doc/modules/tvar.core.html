    result

Tables

tVar
Initialize tVar table

Fields:

  • val (number) value of tVar
  • nameTex (string) LaTeX representation
  • eqTex (string) equation for calculating val i.e. history
  • eqNum (string) equation for calculating val i.e. history with numbers
  • unit (string) is added after result with fixed space
  • numFormat (string) numberformat for displaying val
  • mathEnviroment (string) LaTeX math enviroment that is used for output
  • debugMode (string) if set to "on" every tex.print command is surrounded with \verb||
  • outputMode (string) can be RES for Result, RES_EQ form Result and Equation and RES_EQ_N form Result, Equation and Numbers. Controls the print() command
  • numeration (bool) turns equation numeration on and of. Controls the print() command
  • decimalSeparator (string) sets the decimalSeparatio i.e. ","
  • calcPrecision (number) ammount of decimal places
  • history_fun stores the tVar function for the result
  • history_arg stores the arguments used with history_fun
